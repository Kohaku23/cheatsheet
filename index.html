Tag Name Description
Semantic
<header> Header Defines introductory content or navigational links for a document or section.
<nav> Navigation Contains navigation links, typically used for menus.
<main> Main Represents the primary content area of a webpage.
<section> Section Groups related content together for better organization.
<article> Article Wraps standalone content like blog posts or articles.
<aside> Aside Contains supplementary content, like sidebars.
<footer> Footer Provides a footer section at the bottom of a webpage.
<figure> Figure Wraps media content like images or videos.
<figcaption> Figcaption Provides captions for media content within <figure>.
<time> Time Marks dates, times, or durations for semantic meaning.
Non-Semantic
<div> Division A generic container for grouping and styling elements without conveying specific meaning.
<span> Span Similar to <div>, but used for inline elements and text styling.
Table
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row Defines a row within a table to contain table cells.
<th> Table Header Cell Defines a header cell within a table row, typically used to label columns or rows.
<td> Table Data Cell Defines a standard data cell within a table row, containing data or information.
<tfoot> Table Foot The set of rows defining the footer in a table.
